Getting Started

Start coding with Nova in minutes

1

Install Rust

Nova is built with Rust. Install Rust from rustup.rs

2

Clone Nova

Clone the Nova repository: git clone https://github.com/nova-lang/nova.git

3

Build & Run

Build Nova with cargo build --release and start coding with cargo run

4

Write Code

Create .nova files and run them with cargo run your_file.nova
